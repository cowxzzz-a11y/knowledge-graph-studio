# Knowledge Graph Studio

独立的知识图谱前端项目。

这个目录就是当前唯一保留的图谱 UI 工程，使用 npm 依赖安装 `sigma`、`@react-sigma/core`、`@sigma/utils`，不依赖工作区中的本地 `sigma.js` 或 `relation-graph` 源码目录。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```
